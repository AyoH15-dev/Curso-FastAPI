#---------- MONGODB ----------#
1. En la página buscar MongoDB Community Server, descargar
2. Descargar MongoDB Command Line Database Tools Download

#---------- MONGODB ATLAS ----------#
1. Entrar a la página de MongoDB, iniciar sesión
2. Crear un proyecto
3. En Cluster, crear uno, ir a configuraciones avanzadas
4. Crear usuario y contraseña, y seleccionar My Local Environment y crear cluster
5. En el menú de Clusters, oprimir en Connect, conectar desde VS Code, copiar la URL
6. Ir a VS Code, desconectar la base de datos local, oprimir en Connect with Connection String y pegar la URL
7. Ya conectado a la base de datos, en la página de Mongo, oprimir Go Back y seleccionar Drivers
8. Escoger la versión de Python, copiar la URL y pegar dentro de MongoClient('URL')
9. En Clusters, oprimir en Browse Collections, ahí se muestran las colecciones que creemos
